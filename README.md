# Form Test Project

This project is a simple multi-step form application built to explore and experiment with [shadcn/ui](https://ui.shadcn.com/) components. The goal is to create a clean and modern user interface for a fictional bank's account signup flow.

The project will be under constant development, with plans to add more steps and features in the future.

## Live Version

You can find a live demo of this project here: [https://emiliobeyershelton.github.io/Form-Test/](https://emiliobeyershelton.github.io/Form-Test/)

## Tech Stack

This project is built with the following main technologies:

- **[Next.js](https://nextjs.org/)**: A React framework for building server-side rendered and static web applications.
- **[shadcn/ui](https://ui.shadcn.com/)**: A collection of re-usable UI components.
- **[Redux Toolkit](https://redux-toolkit.js.org/)**: For managing application state.
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework.

## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/EmilioBeyerShelton/Form-Test.git
    cd Form-Test
    ```

2.  **Install dependencies:**
    This project uses `pnpm` as the package manager.

    ```bash
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    pnpm dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Available Scripts

- `pnpm dev`: Runs the app in development mode.
- `pnpm build`: Builds the app for production.
- `pnpm start`: Starts a production server.
- `pnpm lint`: Runs the linter.
